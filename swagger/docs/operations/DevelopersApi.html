This is the API

getMovieById

Returns the movie with the specified ID. Requires authentication.

URL

http://localhost:8080/movies/{id}

HTTP Method

GET

Response Type

MovieDto

Parameters

  • Header: Authorization String

    Authentication header: \"<username> <password>\"

  • Path: id Integer

    ID of the movie to retrieve

getMovies

Returns a list of all movies. Requires authentication.

URL

http://localhost:8080/movies

HTTP Method

GET

Response Type

List

Parameters

  • Header: Authorization String

    Authentication header: \"<username> <password>\"
