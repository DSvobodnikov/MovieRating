This is the API

createMovie

Creates a new movie. Requires authentication and admin privileges.

URL

http://localhost:8080/movies

HTTP Method

POST

Response Type

MovieDto

Parameters

  • Body: body MovieDto(MovieDto)

    Movie to create

  • Header: Authorization String

    Authentication header: \"<username> <password>\" (admin user required)

deleteMovie

Deletes the movie with the specified ID. Requires admin privileges.

URL

http://localhost:8080/movies/{id}

HTTP Method

DELETE

Response Type

Parameters

  • Header: Authorization String

    Authentication header: \"<username> <password>\" (admin user required)

  • Path: id Integer

    ID of the movie to delete

updateMovie

Updates the movie with the specified ID. Requires authentication and admin privileges.

URL

http://localhost:8080/movies/{id}

HTTP Method

PUT

Response Type

MovieDto

Parameters

  • Body: body MovieDto(MovieDto)

    Updated movie data

  • Header: Authorization String

    Authentication header: \"<username> <password>\" (admin user required)

  • Path: id Integer

    ID of the movie to update
